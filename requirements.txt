# SelectTranslate - System Dependencies
#
# This application uses system packages instead of pip packages for better
# stability and integration with the operating system.
#
# INSTALLATION:
# Run: make install
# 
# Or install manually:
#
# Ubuntu/Debian:
# sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-3.0 apertium apertium-eng-spa dict xsel xdotool
#
# Fedora:
# sudo dnf install python3-gobject gtk3-devel apertium dict xsel xdotool
#
# Arch Linux:
# sudo pacman -S python-gobject gtk3 apertium dict xsel xdotool
#
# VERIFICATION:
# Run: make check-deps
#
# DEBIAN PACKAGE:
# Build .deb package: make package
# Install build deps: make install-build-deps
